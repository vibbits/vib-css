Fonts

Regular font for main text content. 
p { font-family: "Open Sans"; }
Font for page and section headings. 
h1 { font-family: "Barlow Condensed"; }
Dense font for links. 
a { font-family: "Dense"; }
Dense bold font for highlighting and emphasis. 
a { font-family: "Dense-Bold"; }
